First Web

This was the first website i ever created during my first year at BMCC in fall 2015. The website was a attempt to create a portfolio style site with a gallery. The site was simple with very little styling.

[Homepage]

The Homepage was the about me page. It had a Author Bio, along with some extra information on the side. At the top there is a h1 tag with my name and under theres a picture of me. The Navigation was all the way at the bottom. I styled each section with diffrent colors so that i would be able to see how far each one expands.

[Assignment]

The Second Page is a assignment page, here i would write all the assignments that were given to me along with their due dates.

[The Gallery]

The last page is a gallery page. For this page i created a couple images that would be placed here. The first image was of a table with a sphere and a cube on top of it. Then a CD cover for a holloween track and lastly a comic book cover. I uploaded them in three diffrent formats png, jpeg and gif.
