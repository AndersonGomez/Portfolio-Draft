Phoebus

Phobus was the first site i used Javascript in. The task was to create 5 diffrent functions using Javascript. It was created in a one page Layout.

[Homepage]

At the top of the page you would see the Navigation Bar. For my first function using Javascript i created a slideshow. The slideshow hosted several images from a music artist.

[The Player]

Under The slideshow was a media player. For my second function i created a Play button and Pause button. The images would change when press to go along with the action.For the Third function i did something simular but for a mute and unmute button.

[The about me page]

The final page was a about me page. The fourth function was a accordian, this was a bar that woould open to reveal text. Finally the last function was a Google Map.
