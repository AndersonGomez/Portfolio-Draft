3 Pins

Spring 2016

These Pins were created during the 2016 presidential campaign. They were created for my Multi Media Art class. These pins were for Bernie Sanders I created three diffrent pins with diffrent design. The first Pin was inspired by a poster that i saw with the squiggly lines. The second Pin i wanted it to be a basic Red, White and Blue color style. And for the last Pin i gave it some of the same design as the first one, bur i used a diffrent font and some lighter colors.
